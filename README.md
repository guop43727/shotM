# MONSTER TIDE - 3D跑酷射击游戏

## 运行方式
直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 游戏特色
- 🎨 赛博朋克霓虹风格UI设计
- 🔫 3D透视跑酷射击视角
- 👾 多种真实怪物图片
- 💥 连续疯狂射击系统
- 🎁 **武器掉落系统** - 击中掉落的武器箱即可获得超级武器！
- 🚪 **数字门机制** - 击中数字门增加数字，碰到门时玩家数量会变成门上的数字！
- 🔥 **动态难度系统** - 获得超级武器或增加玩家数量后，怪物会变得更密集！
- ⚡ 流畅的视觉效果和发光特效

## 游戏玩法
1. 点击"开始波次"按钮开始战斗
2. 机器人会自动向前方的怪物疯狂射击
3. **收集武器箱**获得强力武器（机枪、霰弹枪、激光炮）
4. **射击数字门**增加门上的数字，让它从负数变正数
5. 数字门撞到你时，玩家数量会变成门上的数字
6. **火力越强，怪物越多！** - 挑战你的极限！

## 武器类型
- **步枪** (默认): 标准射速和伤害，火力1x
- **机枪** 🟡: 超快射速，持续10秒，火力2x
- **霰弹枪** 🔴: 一次5发散射子弹，持续8秒，火力2.5x
- **激光炮** 🔵: 超高伤害快速射击，持续12秒，火力3x

## 数字门系统
- 数字门初始为负数（如 -5, -10）
- 每次被子弹击中，数字 +1
- 当数字门到达玩家位置时，玩家数量变成门上的数字
- 策略：尽可能把负数门打成正数，增加玩家数量！
- **玩家数量越多，火力加成越高！**

## 动态难度系统 🔥
- 你的火力等级 = 武器加成 + 玩家数量加成
- 火力越高，怪物生成速度越快，数量越多
- 获得激光炮 + 10个玩家 = 5倍火力 = 怪物潮汹涌而来！
- 挑战：能撑多久？

## 平衡调整
- 初始怪物血量降低（15HP），更容易击杀
- 怪物速度减半，给你足够反应时间
- 密集排队冲锋，每批3-5个怪物

## 游戏目标
- 击杀怪物获得能量点
- 收集武器提升火力
- 操控数字门增加玩家数量
- 在怪物潮中尽可能坚持！

## 技术栈
- HTML5 Canvas 2D渲染
- 3D透视投影算法
- 动态难度调整系统
- 原生 JavaScript
- Google Fonts (Plus Jakarta Sans, Be Vietnam Pro)
- Material Symbols Icons
